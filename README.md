🎬 YouTube Clone – Frontend UI Project
📌 Project Overview

This project is a YouTube Homepage UI Clone built using pure HTML5 and CSS3.
It replicates the layout of YouTube including the navigation bar, sidebar, search section, category buttons, and responsive video grid layout.

The purpose of this project is to practice modern frontend layout structuring, flexbox positioning, and UI replication techniques.

🚀 Features

• Responsive Navigation Bar

• Sidebar with Icons

• Search Bar with Material Icons

• Category / Filter Buttons

• Video Card Layout with Thumbnails

• Clean Grid Structure

• Font Awesome & Google Icons Integration

• External CSS Styling

🛠️ Technologies Used

• HTML5

• CSS3

• Google Material Icons

• Font Awesome
